## Descripción del Proyecto

MyService es una plataforma para clientes con algún problema que resolver (plomería, carpintería, …) y expertos que les ayudan solucionar esos problemas.

## Stack Tecnologias

### Front-End:

- HTML
- CSS
- JavaScript
- React
- Redux Toolkit Query
- TailwindCSS
- Framer Motion
- React Hook
- SweetAlert

### Back-End:

- Java 17
- Springboot
- Spring security 6.0
- Spring data
- MySQL

### Diseño y Organización:

- Figma
- Trello
- Discord

### Deploy:

- Vercel
- Railway
